== Description ==

WPSyncSheets For WooCommerce lite is a powerful export plugin designed to seamlessly migrate <strong>WooCommerce products, orders, customers & coupons to Google Sheets</strong>.

With its user-friendly <strong>'Click to Sync'</strong> feature, you can effortlessly transfer all your product, order, customer, and coupon data into a spreadsheet, making it easy to back up information, migrate data between websites, or share insights with team members and stakeholders for analysis and reporting. 

The WooCommerce exporter plugin organizes your data by sorting orders, products, customers, and coupons into separate sheets within the same Google Sheets document. You have the flexibility to <strong>enable, disable, or create custom sheet headers</strong> and export data into specific sheets tailored to your needs. 

As a powerful <strong>automated and run-time solution</strong>, WPSyncSheets updates your data in real-time, eliminating the need for manual intervention and ensuring that your information is always updated.

= How to Export Orders & Products From WooCommerce (With Images)? =

Exporting WooCommerce orders and products (with images) into Google Sheets using the WPSyncSheets For WooCommerce Lite is easy. 

All you need to do is install and activate the plugin and connect the plugin with your Google GDrive through API integration. Generate a Client ID, Client Secret Key, and Client Token from your Google Clouds Platform account and add it to the plugin.

= WooCommerce Export Orders & Products =

Proceed to the plugin’s Order Settings or Product Settings page, review the various options that allow you to specify what kinds of data you want to export, and then export the data from WooCommerce to Google Sheets. That’s it.

Live demo of exporting orders into Google Sheets: [Click Here](https://demo.wpsyncsheets.com/wpsyncsheets-for-woocommerce/)

<strong>Instructions for testing demo:</strong> 

1. Select 'Add to Cart' → 'View Cart' → 'Proceed to Checkout' button. 
2. Add a (real or fake) billing address and hit the 'Place Order' button 
3. Select 'View Your Orders in Spreadsheet' for sample WooCommerce orders export in Google Sheets

= How to Import Orders & Products From WooCommerce (With Images)? =

To import WooCommerce orders and products (with images) into Google Sheets you need [WPSyncSheets For WooCommerce Pro](https://www.wpsyncsheets.com/wpsyncsheets-for-woocommerce/?utm_source=wp-repo&utm_campaign=wpsyncsheets-for-woocommerce&utm_medium=description). It’s a [paid plugin](https://www.wpsyncsheets.com/pricing/?utm_source=wp-repo&utm_campaign=wpsyncsheets-for-woocommerce&utm_medium=description). 

Install the plugin then add your paid license to activate the paid plugin. Proceed to connect the plugin with your Google GDrive through API integration. Generate a Client ID, Client Secret Key, and Client Token from your Google Clouds Platform account and add it to the plugin.

= WooCommerce Import Orders & Products =

Go to the plugin's Order Settings or Product Settings page, and generate a sample Google Sheets. Add new data, modify or delete data (orders, products, customers, coupons, and events) on the sample Google Sheets, and then proceed to import the data from the spreadsheet. That’s it. 

= How to Export WordPress Users & WooCommerce Customers? =

To export WordPress users and WooCommerce customers into Google Sheets you need the WPSyncSheets For WooCommerce Lite. 

Just install and activate the plugin, and connect it with your GDrive using API integrations. Generate a Client ID, Client Secret Key, and Client Token from your Google Clouds Platform account and add it to the plugin. Then go to the plugin's Customer Settings page, enable the Import Customers option, scroll down, and hit the Import Customer button and that’s it.

= How to Export WooCommerce Coupons? =

To export coupons from WooCommerce to Google Sheets you need the WPSyncSheets For WooCommerce Lite.

All you need to do is install and activate the plugin and connect it with your GDrive using API integrations. Generate a Client ID, Client Secret Key, and Client Token from your Google Clouds Platform account and add it to the plugin. Then go to the plugin's Coupon Settings page, enable the Import Coupons option, scroll down, and hit the Import Coupons button and that’s it.

== Frequently Asked Questions ==

= Can the plugin export WooCommerce products with images? =

Yes, the plugin can export WooCommerce products with images from your WooCommerce store to Google Sheets. 

= Does the plugin support the export of variation products? =

Yes, the plugin supports the export of variation products from your WooCommerce store into Google Sheets. 

= Does the plugin support multisite? =

Yes, the plugin supports multisite exports to Google Sheets.

= Does the plugin export WooCommerce customers and coupons? =

Yes, the plugin can export WooCommerce customers and coupons into Google Sheets. 

= Does the plugin export posts, pages, and custom post types? =

No, the plugin does not export posts, pages, and custom post types. However, we have a different plugin called [WPSyncSheets For Core](https://www.wpsyncsheets.com/wpsyncsheets-for-core/?utm_source=wp-repo&utm_campaign=wpsyncsheets-for-woocommerce&utm_medium=description) that can export those in Google Sheets. 

= Can the plugin import orders in WooCommerce? =

No, it’s a WooCommerce exporter plugin and cannot import orders in WooCommerce. However, we have a paid plugin called [WPSyncSheets For WooCommerce Pro](https://www.wpsyncsheets.com/wpsyncsheets-for-woocommerce/?utm_source=wp-repo&utm_campaign=wpsyncsheets-for-woocommerce&utm_medium=description) that allows the import of orders into WooCommerce. 

= Can the plugin import WooCommerce product? =

No, it’s a WooCommerce exporter plugin and cannot import products in WooCommerce. However, we have a paid plugin called [WPSyncSheets For WooCommerce Pro](https://www.wpsyncsheets.com/wpsyncsheets-for-woocommerce/?utm_source=wp-repo&utm_campaign=wpsyncsheets-for-woocommerce&utm_medium=description) that allows the import of orders into WooCommerce.

== Installation ==

1. Upload the entire `wpsyncsheets-woocommerce` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **WPSyncSheets Lite For WooCommerce** menu in your WordPress admin screen.

For basic usage, have a look at the [plugin's Documentation](https://docs.wpsyncsheets.com/wpssw-introduction/).
